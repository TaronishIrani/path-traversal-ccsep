Path Traversal Test Site Home Page

This is a very basic website to use for testing path traversal exploitation and patching. Path Traversal works very similiarly to a SQL Injection attack, so the user input box below can be used to test if an attack can be successful.

I recommend adding a few test files in your webroot folder which should be /var/www/pathtraversal/html/... to see if you can access them or not.
